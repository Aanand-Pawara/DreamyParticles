ğŸš€ Getting Started

Prerequisites

All you need is a modern web browser. No Node.js or build steps required.

Installation

Clone the repository

git clone [https://github.com/AanandPawara/dreamy-particles.git](https://github.com/AanandPawara/dreamy-particles.git)


Open the project
Simply open particle_release.html (or index.html) in your browser.

Optional: For the best experience, run it using a local server (like Live Server in VS Code) to avoid any local file security restrictions with canvas data, though it generally works standalone.
